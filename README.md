# Premier League Title Predictor

A machine learning project that predicts Premier League outcomes and simulates the final league table as the season progresses.

## Tech Stack
- Python
- pandas
- numpy
- scikit-learn
- requests
- matplotlib
- Streamlit

## Project Goals
- Fetch Premier League data from API
- Build team-level features
- Predict match outcomes
- Simulate the rest of the season
- Estimate title probabilities and final league positions
